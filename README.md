# Darts Checkout Calculator

## Description
The **Darts Checkout Calculator** is an Android application that helps dart players quickly calculate the best checkout combinations from any given score.  
This app was developed as a **portfolio project** to showcase practical Android development skills using Kotlin.

## Key Features
- Quickly calculates checkout combinations  
- Supports all standard dart scores  
- Simple and intuitive user interface  
- Works offline (no internet connection required)

## Technologies Used
- **Android Studio** – Development IDE  
- **Kotlin** – Programming language  
- **XML Layouts** – User interface design
